# Poké Fresh - E-commerce de Comida Hawaiana 

Este proyecto es una aplicación web full-stack que simula un sistema completo de pedidos de comida. Desarrollado para el ramo **Desarrollo Web y Móvil** de la Universidad Andrés Bello.

## Descripción del Proyecto
La aplicación gestiona todo el ciclo de vida de una compra:
* **Módulo de Cliente:** Registro de usuario, inicio de sesión y proceso de compra interactivo.
* **Módulo de Administración (Vista Chef):** Panel exclusivo para la gestión y visualización de pedidos entrantes, implementando control de roles.

## Tecnologías Utilizadas
* **Frontend:** JavaScript (Vanilla), HTML5, CSS3.
* **Backend:** Node.js, Express.
* **Base de Datos:** MongoDB.
* **Control de Versiones:** Git & GitHub.

## Funcionalidades Destacadas
* **Autenticación:** Creación y validación de cuentas de usuario.
* **Arquitectura de Roles:** Diferenciación clara entre la interfaz de usuario y el panel administrativo/chef.
* **Persistencia:** Almacenamiento dinámico de datos de usuarios y pedidos en tiempo real.
